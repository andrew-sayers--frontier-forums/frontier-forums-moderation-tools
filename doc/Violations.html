List of known violations

maintains a list of violations. If the variables 'PM-worthy violations', 'warning-worthy violations' and/or 'infraction-worthy violations' exist, the names in those lists are used to set the default_user_action for each variable.
Example
var vi = new Violations({
     v                  : v,
     bb                 : bb,
     cache              : violations_cache,
     cache_updater      : function(cache) { ... },
     reload_interval    : 60 * 1000,
     error_callback     : handle_error,
     default_user_action: 'PM' // or 'warning' or 'infraction'
});
