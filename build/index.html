Frontier Forums Moderation Tools

If you've installed a user script extension (e.g. Greasemonkey or Tampermonkey), install the user script however your extension tells you (usually just by clicking the link).

To install in Firefox, click here.

To install in Chrome...

  1. right-click here and save the file
  2. Type chrome://extensions into the URL bar and press enter
  3. Drag and drop the extension on to the page
If Chrome automatically disables the extension, you will need to perform these extra steps:
  1. download Chrome group policy templates
  2. copy [zip]\windows\chrome.admx to c:\windows\policydefinitions
  3. copy [zip]\windows\[yourlanguage]\chrome.adml to c:\windows\policydefinitions\[yourlanguage]\chrome.adml (not c:\windows\[yourlanguage])
  4. type chrome://extensions/ in Chrome's URL bar and press enter
  5. check the Developer Mode checkbox at the top
  6. scroll down the list of disabled extensions and note the ID of the Frontier Forums Moderation Tools extension. At the time of writing, this was: nndcellimmcbpaggigggceekfaepkjbd
  7. open the Run window, either by clicking Start -> Run or by pressing the Windows key and r
  8. type gpedit.msc
  9. press OK
  10. if gpedit.msc is not found, install it using these instructions
  11. expand User Configuration -> Administrative Templates -> Google -> Google Chrome -> Extensions
  12. double-click to open Configure extension installation whitelist
  13. select Enabled, then click Show...
  14. in the list, enter the ID for the extension
  15. click OK and restart Chrome
