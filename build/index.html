Frontier Forums Moderation Tools (beta)

Feature list:

  • A lot of configuration has been moved into test thread 32626
    • Note: the previous version let anyone define their own template, but changing the defaults meant editing the source,
      This version lets any moderator change the defaults, but making a private variation means editing the source.
  • When viewing profiles in ModCP
    • "go to member page" and "send PM" buttons
      note: doesn't work in Chrome when you go to /modcp/user.php - fixing that would be a long-term project
  • When moderating posts
    • Validating threads hides the "validate posts" section
    • posts are coloured to indicate whether they will be validated/deleted/ignored
    • One-click checking on StopForumSpam.com
    • Count of modified posts at the bottom
    • buttons now have more descriptive text
  • On thread pages
    • Watch list
    • thread's moderation log, list of active reports and most recent post in the mod log appear at the bottom
      note: reported posts aren't styled correctly - this might be fixed when the forums go live, otherwise I'll do it
    • "Report post" text changed for already-reported posts
    • Single click to report an issue of a given type, go to the report and take it
    • One-click checking on StopForumSpam.com
      Note: seems broken in Chrome right now due to security issues - this will probably fix itself when we go live
  • On report pages:
    • One-click button to take a thread
    • Data shown from the reported user's member page, profile and user notes
    • Context-dependent common actions
    • Notification about reports for threads that have now been closed
    • For open threads owned by you, "common actions" guides you through the whole reporting process
      note: the action form will be finalised once we have final confirmation about how the process will work in future
  • On edit pages
    • Delete message is simply marked "recommended" now the interface is less ambiguous

Features removed in the new version:

  • Notes to self (useless)
  • Automatic sig violation detection (redundant)
  • Automatic swear detection (redundant)
  • Templates (replaced with variables and overhauled reporting system)
  • Mollyguard on the "physically remove" button on edit pages (redundant)

To install the beta in Firefox, click here

To install in Opera/Chrome...

  1. right-click here and save the Chrome extension
  2. type chrome://extensions into the URL bar and press enter
  3. drag and drop the extension on to the page
If Chrome automatically disables the extension, try this alternate solution. It should work as normal, but will cause Chrome to complain about this extension running in developer mode:
  1. click here to download the zipped Chrome extension
  2. copy the frontier-forums-moderation-tools folder to an appropriate place on your hard drive (e.g. your Documents folder)
  3. type chrome://extensions into the URL bar and press enter
  4. check the Developer Mode checkbox at the top
  5. click Load unpacked extension...
  6. open the frontier-forums-moderation-tools folder you previously saved
